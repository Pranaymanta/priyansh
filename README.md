# FoodHub Application

## Project Structure
- `frontend/`: Contains the React frontend code.
- `backend/`: Contains the Node.js backend code.
- `database/`: Contains SQL files for database structure.

## Features
- Full-stack application with Supabase integration.
- Real-time tracking and notifications.
- Authentication.

## To get started
1. Clone the repository.
2. Navigate to the frontend and backend directories, and install dependencies using `npm install`.
3. Set up environment variables for Supabase.
4. Run the backend server and frontend client.

## Database Schema
- Users
- Orders
- Products

## API Routes
- `POST /api/auth/signup`
- `POST /api/auth/login`
- `GET /api/orders`
- `POST /api/orders`
